# computer-science-quiz
A Computer Science related set of Quiz which includes 5 subjects, 1. python, 2. DSA, 3. OS, 4.DBMS and 5. Computer Networks. A python based quiz-project runs on terminal, questions are fetched using API.
A simple Python-based multiple-choice quiz that fetches computer science questions from the Open Trivia Database API.

## Features
- Fetches random computer science questions.
- Multiple-choice answers.
- Shows score and percentage at the end.
  

## How to run
1. Make sure Python 3 is installed
2. Run the script:
```bash
python cse-quiz.py
